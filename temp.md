增加一个功能，可以自动检测 outline 的层级来决定 header 的数量，举个例子

输入这个内容
```  
- 1
    - 1.1
    - 1.2
        - 1.2.1
            - 1.2.1.1
        - 1.2.2
- 2
    - 2.1
        - 2.1.1
- 3
    - 3.1
```  
应该转换为 
```  
## 1
### 1.1
### 1.2
- 1.2.1
    - 1.2.1.1
- 1.2.2
## 2
- 2.1
    - 2.1.1
## 3
- 3.1
```

你自己看一下，举个例子，如果这里 3.1 也被转换为 “### 3.1” ，会很奇怪，因为这样 3.1 下面其实是没有内容的，不足以单独成为一个标题

建议你创建一个新的函数，跟原来的转换层级的方式区分开，并且给用户一个新的选择，同时建议你也添加一些单元测试
